# Proyecto ThinkusAPI

Este proyecto es una aplicación basada en C# que utiliza MongoDB para almacenar y administrar información sobre pólizas de seguros.

## Pasos de configuración

1. Crea una colección en MongoDB para almacenar los datos de la base de datos. 

2. Asigna el valor "Insurance" a la colección creada.

3. Crea una colección en la base de datos llamada "Polizas".

4. Descarga y abre la solución ThinkusAPI en tu entorno de desarrollo.
